# Task 3 - Data Visualization & Dashboarding

## Project Overview

This project was completed as part of the ApexPlanet Data Analytics Internship Program.

The objective of this task was to create a business dashboard using the Superstore Sales dataset and present key insights through visual analytics.

## Project Title

**Superstore Sales Dashboard**

## Tools Used

* Python
* Pandas
* Matplotlib / Plotly
* Superstore Sales Dataset

## Dashboard Features

The dashboard includes:

* **Total Sales KPI**
* **Total Profit KPI**
* **Total Quantity KPI**
* **Sales by Category**
* **Profit by Category**
* **Region-wise Sales**
* **Monthly Sales Trend**
* **Top 10 Customers by Sales**

## Key Insights

* Category-wise comparison of sales and profit
* Region-wise contribution to overall sales
* Monthly sales trend analysis
* Identification of top customers based on sales
* Overall business performance using KPI metrics
